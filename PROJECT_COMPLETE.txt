🚀 QUICK START
═══════════════════════════════════════════════════════════════════════
1. Install dependencies:
   pip install -r requirements.txt
2. List available firms:
   python main.py --list-firms
3. Configure your challenge in main.py:
   CONFIG = {
       'firm_key': 'funded_next_6k',
       'starting_balance': 6000.0,
       'magic_filter': None,
       'auto_close_on_breach': False,
       'poll_interval': 1.0,
       'log_level': 'INFO',
   }
4. Start MT5 and login to your account
5. Run the monitor:
   python main.py

python main.py --log-level DEBUG           # Debug logging

🔍 RULES MONITORED
═══════════════════════════════════════════════════════════════════════
✓ Max Daily Loss (5%)         - Equity vs. day's opening balance
✓ Max Total Drawdown (10%)    - Static or trailing calculation
✓ Profit Target (10%)         - Balance + minimum trading days
✓ Trading Days                - Days with closed trades
✓ Challenge Time Limit        - Calendar days (usually 30-60)
✓ Lot Size Limits             - Per trade and total (if configured)
